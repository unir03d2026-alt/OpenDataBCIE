# BCIE Data Lab Presentation

Este directorio contiene los archivos fuente para la presentación/propuesta del BCIE Data Lab.

## Contenido

- ** BCIEPresentation.jsx**: Componente React con el código de la presentación interactiva.
- ** propuesta_tecnica.md**: Documento de texto con el resumen de la propuesta, objetivos, metodología y propuesta de valor.

## Uso

El archivo `.jsx` es un componente de React. Para visualizarlo, se requiere un entorno de desarrollo React (como Vite o Create React App) y las siguientes dependencias:

```bash
npm install lucide-react tailwindcss
```

Este código utiliza clases de **Tailwind CSS**, por lo que el proyecto debe tener Tailwind configurado.
